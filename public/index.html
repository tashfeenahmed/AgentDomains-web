free domains for AI agents

Your agent built a site.
Give it a domain.

yourname.makes.fyi

When an agent ships a website or an API, it needs somewhere to put it. One command gets a working domain that the agent can wire up by itself.

Get started → ★ GitHub
1

Sign up

$ agentdns signup
2

Claim a domain

$ agentdns claim yourname
3

Point it anywhere

→ yourname.makes.fyi ✨
🔑 works without an email 🤖 --json for scripts 🔒 bring your own SSL 🌐 use your own nameservers
# install
$ go install github.com/tashfeenahmed/AgentDNS/cmd/agentdns@latest

# point a domain at the site your agent just built
$ agentdns claim mybot --type A --content 203.0.113.10
✓ mybot.makes.fyi is live
